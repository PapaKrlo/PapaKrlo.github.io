This Trainer Does Not Exist



Python

HTML, CSS JavaScript

StyleGAN-2ADA

Pytorch

I created a website as part of a university module that produced AI-generated images of trainers using PyTorch and StyleGAN-2ADA. The website allowed users to click a button to cycle through AI-generated images. I used a dataset of more than 500 trainer images to train the model. With the help of a pre-trained model from a GitHub repository, I trained the AI model for 105 ticks, which equates to about 61 hours. I initially used a DCGAN (Deep Convolutional Generative Adversarial Network) but later discovered that StyleGAN-2ADA produced higher-quality images. The resulting AI-generated images were indistinguishable from real ones, meeting the expectations of a network like StyleGAN-2ADA. I created the website using Code Sandbox to demonstrate the real-life application of this project, which is intended for trainer enthusiasts and shoe designers seeking inspiration for new designs.
